This is the new page of the JSafran software, with a Java Web Start link to automatically install JSafran on your computer, along with all its dependencies.
The previous installation mode is more complex (see old JSafran page) because it requires you to download and uncompress all resources manually.
This Java Web Start solution should be way easier for you to test JSafran. If you are interested by the source code, I distribute it freely through a public GIT repo (see the old page above), which pitifully is never up-to-date. I have a private GIT repo with the last (unstable) version. Please contact me if you want to access it.

Clic here to install and run JSafran:jsafran


The installer will be put a few files (less than 10) and 2 directories in your HOME directory.

Warning ! Installation will download about 150Mb from the internet, so please be sure to have a good enough connection, and at least 200Mb free space on your disk ! It also requires at least 1Gb of RAM. You also have to have a java 1.6 plugin installed in your browser. If your browser does not launch automatically the .jnlp file, then you can also download the .jnlp file and run it from the command-line with javaws.

The installer will warn you that JSafran may be harmful to your computer: this is a standard safeguard warning for all Java Web Start applications. You can safely accept it: I am a researcher working for a public French research institute (CNRS) and within a recognized public laboratory (LORIA): I have no interest in delivering malicious software. Furthermore, all my code is open-source, so you can check it, and I guarantee that the code is free of any type of ad-ware, spy-ware, virus, etc.

Latest News

JSafran includes now both English and French models (Tree tagger and MATE parser).
The English parsing models have been trained on the Conll'08 training corpus. These models give LAS=85.8% and UAS=88.1% on the test-WSJ of Conll'08, with POS-tags obtained from the TreeTagger.
There are two French models: one trained on the French Treebank (written text), and another trained on the Ester Treebank (speech transcripts).
